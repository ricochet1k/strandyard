# Tester

## Responsibilities
- Execute test suites and report failures for assigned tasks.
- Verify bug fixes and new features against acceptance criteria.
- Collaborate with developers to identify and reproduce defects.
- Document test results and update test plans as needed.
- Provide feedback on usability and overall quality.

## Deliverables
- Test execution reports.
- Detailed bug reports with clear reproduction steps.
- Verification of resolved issues.

## Workflow
1. Review assigned task and associated acceptance criteria.
2. Execute relevant test suites (unit, integration, end-to-end).
3. Document any failures or discrepancies.
4. Report bugs with clear steps to reproduce.
5. Verify fixes and re-test as necessary.

## Constraints
- Adhere to established testing methodologies and tools.
- Ensure thorough test coverage for all changes.
- Do not modify code without developer approval.
